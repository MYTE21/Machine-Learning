# Calculus

`The word Calculus comes from Latin meaning "small stone".`

**Calculus -** Calculus is the study of how things change. It provides a framework for modeling systems in which there is change, and a way to deduce the predictions of such models.

![Calculus - Differential and Integral](/assets/images/calculus/kinds_of_calculus.png "Calculus - Differential and Integral")

* **Differential Calculus :** Cuts something into small pieces to find how it changes.

* **Integral Calculus :** Joins (integrates) the small pieces together to find how much there is.

**Variable :** In mathematics, a variable is a symbol used to represent an arbitrary element of a set. Generally "x, y, z, u, v, w" etc. letters are used for variables.

**Constant :** A fixed value. 1, 2, 3, <img src="https://latex.codecogs.com/svg.latex?\pi"/>, ... etc. are constant. "a, b, c, d, ... <img src="https://latex.codecogs.com/svg.latex?\alpha"/>, <img src="https://latex.codecogs.com/svg.latex?\beta"/>, <img src="https://latex.codecogs.com/svg.latex?\gamma"/>," are used for constants.

**Continuous variable :** If `x` contains all the real number from `a` to `b` continuously then `x` is called continuous variable. Then the domain of `x` is `a`<img src="https://latex.codecogs.com/svg.latex?\leq"/>`x`<img src="https://latex.codecogs.com/svg.latex?\leq"/>`b`.

If `a` doesn't included in the domain of `x` then it'll be `a`<img src="https://latex.codecogs.com/svg.latex?<"/>`x`<img src="https://latex.codecogs.com/svg.latex?\leq"/>`b`. Here the `x` is open in the left side, close in the right side.

`a`<img src="https://latex.codecogs.com/svg.latex?\leq"/>`x`<img src="https://latex.codecogs.com/svg.latex?<"/>`b` , here left is closed and right is open.

`a`<img src="https://latex.codecogs.com/svg.latex?<"/>`x`<img src="https://latex.codecogs.com/svg.latex?<"/>`b`, here both side is open.
