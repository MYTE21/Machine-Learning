# Calculus

`The word Calculus comes from Latin meaning "small stone".`

**Calculus - Calculus is the study of how things change. It provides a framework for modeling systems in which there is change, and a way to deduce the predictions of such models.**

![Calculus - Differential and Integral](/assets/images/calculus/kinds_of_calculus.png "Calculus - Differential and Integral")

* **Differential Calculus :** Cuts something into small pieces to find how it changes.

* **Integral Calculus** Joins (integrates) the small pieces together to find how much there is.
